This folder contains various packages for this project.


embed-code
----------

This directory includes an EMBED.html file which contains a code you can use to embed this project within your CMS.


public
------

A minified version of the embeddable page with all necessary assets which you can host on your own server.


source
------

An UN-minified version of the project, which includes the Reuters Graphics page (index.html) and the embeddable page (embed.html), which you can customize and host on your own servers.


app.zip
-------

This archive includes the full development environment used to create the project, with all source files used to build the page. This archive is for developers and will require installing custom software on your machine.

If you'd like to use this archive to customize and rebuild the page to your specifications, visit the instructions for the Reuters Graphics rig here: https://reuters-graphics.github.io/style/graphics-rig/

The quickstart guide will get you up and running.

Please be aware: The documentation is designed for Reuters Graphics staff so some functionality will not be accessible to you, for example, instructions for publishing a project to the Reuters Graphics servers.
